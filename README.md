# CIFAR-10 Image Classification using CNN

## Overview
This project implements a Convolutional Neural Network (CNN) to classify images from the CIFAR-10 dataset into 10 categories.

## Technologies Used
- Python
- TensorFlow / Keras
- Google Colab

## Model Performance
- Training Accuracy: ~79%
- Validation Accuracy: ~72%

## Features
- CNN-based image classification
- Data preprocessing & normalization
- Model training & evaluation
- Saved trained model

## Files Included
- notebook.ipynb
- cnn_model.keras
- report.pdf

## Result
The CNN model successfully classifies images with good accuracy and generalization.
